## WIX STUDIO PAGE CREATION WORKFLOW

### Repeatable Template Established

**Step 1: Create Page**
- Click "+Add New Page" in Site Pages panel
- Select "Page" (blank template)

**Step 2: Rename**
- Double-click page name
- Type new name
- Press Enter

**Step 3: Configure SEO**
- Right-click page name
- Click "SEO Basics"
- Update URL slug (lowercase, hyphens)
- Update Title tag (keyword + value + brand)
- Update Meta description (150-160 chars, keyword-focused)
- Verify indexing toggle is ON

**Step 4: Add Content** (after all 7 pages created)
- Add heading with keyword
- Add body content
- Add CTA button
- Add internal links to hubs
