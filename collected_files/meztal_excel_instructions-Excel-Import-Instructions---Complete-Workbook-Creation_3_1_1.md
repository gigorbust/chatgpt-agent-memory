# MEZTAL MASTER SEO STRATEGY - EXCEL IMPORT INSTRUCTIONS

## Quick Setup (5 Minutes)

### Step 1: Create New Excel Workbook
1. Open Microsoft Excel
2. Create a new blank workbook
3. Save as "MEZTAL_MASTER_SEO_STRATEGY.xlsx"

### Step 2: Import Each CSV Tab
**For each of the 5 CSV artifacts above:**

1. **Create New Tab**
   - Right-click on sheet tabs at bottom
   - Select "Insert Worksheet" 
   - Rename to match tab name (e.g., "Master_Keyword_Inventory")

2. **Import CSV Data**
   - Click on the CSV artifact above
   - Select ALL content (Ctrl+A)
   - Copy (Ctrl+C) 
   - Go to Excel tab
   - Click cell A1
   - Paste (Ctrl+V)
   - Excel will automatically separate into columns

3. **Format Headers**
   - Select row 1 (headers)
   - Make bold (Ctrl+B)
   - Add background color (optional)
   - Apply borders

### Step 3: Set Up Filtering & Sorting
**For each tab:**
1. Select all data (Ctrl+A)
2. Go to Data → Filter
3. This adds dropdown arrows to headers
4. Now you can sort and filter by any column

### Step 4: Set Column Widths
**Auto-resize all columns:**
1. Select all (Ctrl+A)
2. Double-click any column border
3. All columns auto-resize to content
